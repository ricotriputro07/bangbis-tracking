📋 Cara Setup
1. Buka Google Sheets baru
2. Menu Extensions → Apps Script
3. Paste kode dari file google_apps_script_api.js
4. Deploy → New Deployment → Web App
    Execute as: Me | Who has access: Anyone
5. Copy URL → paste di kolom di atas
6. Jalankan fungsi setupFormatting() di editor
7. Klik Import Data untuk migrasi data
